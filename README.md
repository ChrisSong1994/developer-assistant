开发者助手

- [x] 颜色拾取和转换

- [ ] 正则工具

  - [ ] 非对称加解密
  - [ ] hash 加密
- [ ] 编码转换
- [ ] 图片处理
- [ ] 简单接口请求
- [ ] diff 文本
- [ ] 文件压缩
- [ ] 代码执行 vm

## setup

### install

```sh
cd developer-assistant
yarn
```

### start

```sh
yarn run dev
```

### build

```sh
yarn run electron:build:mac  # for mac
```
